## Pinball Arcade

Se incluye un sencillo juego de pinball escrito en Python con `pygame`.

### Requisitos

- Python 3.10+
- Dependencias instalables con `pip install -r requirements.txt`

### Controles

- Flecha izquierda / **A**: Activa el flipper izquierdo
- Flecha derecha / **D**: Activa el flipper derecho
- **Espacio**: Lanza una nueva bola cuando no hay una en juego
- **Esc** o **Q**: Salir

### Ejecución

```bash
python pinball_game.py
```

Un rectángulo en el lado derecho funciona como carril de lanzamiento. El marcador se incrementa al golpear los bumpers repartidos por la mesa.
